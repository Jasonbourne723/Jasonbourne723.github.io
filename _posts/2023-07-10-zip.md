## tar

### 打包并压缩

```
tar -zcf a.tar.gz a.txt
```

### 解压

```
tar -zxvf a.tar.gz
```

## zip/unzip

### 安装zip/unzip

```
yum install zip unzip -y
```

### zip压缩

```
zip -r a.zip a.txt
```

### unzip解压

```
unzip a.zip
```
